# Node.js Car Rental App

A simple car rental management system built with Node.js and Express.

##  Features
### User
- View list of available cars
- Reserve / order a car

### Admin
- Add new cars
- Edit car details
- Delete cars
- Manage fleet

##  Tech Stack
- Node.js
- Express.js
- MongoDB / MySQL (depending on your version)
- EJS / Handlebars (if using server-side views)

##  Run the project

```bash
npm install
npm start
